# Reviewer Guide

## Reviewer Job

Reviewers decide whether an asset can be used publicly, internally, archived, or blocked.

AI and uploaders may suggest tags. They do not approve rights.

## Required Before Approval

Every approved asset needs:

```text
title
visible_tags
TJC_terms
quality_status
rights_status
publish_status
usage_scope
reviewed_by
reviewed_date
approval_notes
```

## Decision States

| State | Use when |
|---|---|
| Approved Public | Safe for public TJC use. |
| Approved Internal | Safe inside TJC, not public. |
| Searchable Archive | Useful to find later, but not publishable yet. |
| Archive - Not Promoted | Safe enough to keep, low stock-media value. |
| Do Not Use | Rights, privacy, safety, or mission concern. |
| Needs Review | Default for anything uncertain. |

## Sensitive Media

Keep these in `Needs Review` unless clearly approved:

- minors/children
- unknown people
- baptism
- footwashing
- holy communion
- worship footage
- sermons
- music, hymns, backing tracks
- third-party design assets
- volunteer-created media with unclear permission

## Approval Rule

Preview success is not approval. Good-looking media can still be unsafe to publish.

When unsure, choose:

```text
Needs Review / Do Not Publish
```
